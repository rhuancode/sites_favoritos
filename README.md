# 🌐 Meus Sites Favoritos

Projeto desenvolvido como atividade da disciplina de **Programação Web I**, com o objetivo de aplicar conceitos básicos de HTML5.

---

## 📌 Descrição

Esta página web apresenta uma lista de 5 sites favoritos, contendo:

* Nome de cada site
* Imagem representativa
* Link clicável que direciona para o site

O projeto foi desenvolvido utilizando apenas **HTML**, sem uso de CSS ou JavaScript, conforme os conteúdos estudados até o momento.

---

## 🖼️ Funcionalidades

* Exibição de imagens locais
* Uso da tag `<figure>` para organização semântica
* Uso da tag `<figcaption>` para legenda
* Links externos com abertura em nova aba (`target="_blank"`)
* Estruturação com títulos (`<h1>` e `<h2>`)
